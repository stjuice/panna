# PWA icons

Add these files for the "Add to Home Screen" / install app icon:

- **icon-192.png** – 192×192 px
- **icon-512.png** – 512×512 px

You can export them from `../icon.svg` using an image editor, or use a tool like [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator) or [realfavicongenerator.net](https://realfavicongenerator.net/).
